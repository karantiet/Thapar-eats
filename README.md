# UI/UX Elective Project: Food Order Landing Page

## Project Overview

This project, undertaken as part of my 5th-semester coursework in the UI/UX elective, involves the creation of a responsive webpage. The focus is on designing a Food Order Landing Page using HTML, CSS, and JavaScript to elevate the overall user experience.
